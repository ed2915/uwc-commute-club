Privacy and consent

UWC Commute Club collects your random 4-digit code, direction, suburb, and selected travel times only to group route/time interest.

Providing a random code is voluntary, but it is needed to join route pools and avoid duplicate entries. Your route interest will be kept only for this commute-club project and should be deleted when it is no longer needed for that purpose.

You can remove your code and all route-pool entries linked to it at any time using the remove button below. When you remove it, the code is deleted from the active database, and no extra copies are kept by this app that would continue to store it.

Your code will not be shared with anyone in a pool unless you give explicit consent at a later stage.

Public heatmaps show aggregate counts only. Admin access is token-protected.

To remove your code from the database, enter it above and use the remove button. This deletes all route-pool entries linked to that code.
